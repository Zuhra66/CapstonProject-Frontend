# EmpowerMedWellness – Frontend

Frontend application for **EmpowerMedWellness**, a holistic wellness platform built as a capstone project.  
This React application provides user-facing and admin interfaces for wellness services, events, education, memberships, blogs, and products.

## Project Overview

The EmpowerMedWellness frontend delivers a responsive, user-friendly experience that allows users to explore wellness resources and enables administrators to manage platform content securely.

---

##  Tech Stack

- **React**
- **Vite**
- **JavaScript (ES6+)**
- **CSS Modules & Bootstrap**
- **Auth0** (authentication)
- **Fetch API**
- **Deployment**: Render

# Run Locally
npm install
npm run dev
